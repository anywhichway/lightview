Template Components

You can use templates to define components. Note how the `mount` function is defined during component creation and the script containing it does not need to have the id `lightview`. Try and change the tag name to something else. (Hint: You do not need to change the id of the template. And, you will need to make changes to both the `head` and the `body`). See the API documentation on [Lightview.createComponent](../api.html#createComponnent).
