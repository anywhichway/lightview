Progress

Progress bar shows the progression of a task.

Quick Example

import Progress from './components/data-display/progress.js';

// Using Lightview component
Progress({ value: 40, max: 100 })

// Or using DaisyUI classes directly
<progress class="progress w-56" value="40" max="100"></progress>

Props

Prop Type Default Description
value number | signal 0 Current value
max number 100 Maximum value
color string - 'primary' | 'secondary' | 'accent' | 'success' | 'warning' | 'info' | 'error'

Colors

Indeterminate

Reactive Example

const { signal, tags } = Lightview;
const { div, button } = tags;
const Progress = LightviewX.getComponent('Progress');

const progress = signal(0);
setInterval(() => {
    progress.value = (progress.value + 10) % 110;
}, 500);

const demo = div({ class: 'flex flex-col gap-4' },
    Progress({ 
        value: progress, 
        max: 100, 
        class: 'progress-primary w-full' 
    }),
    div(() => `Loading... ${progress.value}%`)
);
For more examples and options, see the DaisyUI Progress documentation.
