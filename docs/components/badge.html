Badge

Badges are used to inform the user of the status of specific data.

Quick Example

Badge Primary Secondary 
import Badge from './components/data-display/badge.js';

// Using Lightview component
Badge({ color: 'primary' }, 'Primary')

// Or using DaisyUI classes directly
<span class="badge badge-primary">Primary</span>

Props

Prop Type Default Description
color string - 'primary' | 'secondary' | 'accent' | 'success' | 'warning' | 'info' | 'error' | 'ghost' | 'neutral'
size string 'md' 'xs' | 'sm' | 'md' | 'lg'
variant string - 'outline' | 'soft' | 'dash'

In Button

Reactive Example

const { signal, tags } = Lightview;
const { div, button } = tags;
const Badge = LightviewX.getComponent('Badge');

const count = signal(0);

const demo = div({ class: 'flex items-center gap-4' },
    button({ class: 'btn btn-sm', onclick: () => count.value++ }, 'Increment'),
    div({ class: 'relative inline-block' },
        'Messages',
        Badge({ 
            class: 'badge-xs badge-primary absolute -top-2 -right-3',
        }, () => count.value)
    )
);
For more examples and options, see the DaisyUI Badge documentation.
