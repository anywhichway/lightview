Using Components

Two ways to use:

1. JavaScript Components

import { Button, Card } from './components/index.js';

const App = Card({ shadow: 'md' },
    Card.Body({},
        Button({ color: 'primary' }, 'Click me')
    )
);

2. DaisyUI Classes

<div class="card bg-base-100 shadow-md">
    <div class="card-body">
        <button class="btn btn-primary">
            Click me
        </button>
    </div>
</div>
