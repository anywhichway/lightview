File Input

File Input allows users to upload files.

Quick Example

import FileInput from './components/data-input/file-input.js';

// Using Lightview component
FileInput({ bordered: true, accept: 'image/*' })

// Or using DaisyUI classes directly
<input type="file" class="file-input file-input-bordered" />

Props

Prop Type Default Description
accept string - Accepted file types
multiple boolean false Allow multiple files
color string - 'primary' | 'secondary' | 'accent' | 'success' | 'warning' | 'info' | 'error' | 'ghost'
size string 'md' 'xs' | 'sm' | 'md' | 'lg'
bordered boolean true Add border
