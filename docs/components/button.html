Button

Buttons allow the user to take actions or make choices. Available in multiple variants, colors, and sizes.

Quick Example

import Button from './components/actions/button.js';

// Using Lightview component
Button({ color: 'primary' }, 'Primary')

// Or using DaisyUI classes directly
<button class="btn btn-primary">Primary</button>

Props

Prop Type Default Description
color string - 'primary' | 'secondary' | 'accent' | 'neutral' | 'info' | 'success' | 'warning' | 'error' | 'ghost' | 'link'
size string 'md' 'xs' | 'sm' | 'md' | 'lg'
variant string - 'outline' | 'soft' | 'dash' | 'wide' | 'block' | 'square' | 'circle'
disabled boolean | function false Disable the button (reactive)
loading boolean | function false Show loading state (reactive)
active boolean false Force active/pressed state
glass boolean false Glass morphism effect

Reactive Loading State

const { signal, tags } = Lightview;
const { button, span } = tags;

const isLoading = signal(false);

const LoadingButton = button({
    class: 'btn btn-primary',
    onclick: async () => {
        isLoading.value = true;
        await new Promise(r => setTimeout(r, 2000));
        isLoading.value = false;
    }
},
    () => isLoading.value 
        ? [span({ class: 'loading loading-spinner' }), ' Saving...']
        : 'Save Changes'
);
For more examples and options, see the DaisyUI Button documentation.
