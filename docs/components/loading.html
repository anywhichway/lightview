Loading

Loading shows an animated indicator for loading states.

Quick Example

import Loading from './components/data-display/loading.js';

// Using Lightview component
Loading({ type: 'spinner', size: 'md' })

// Or using DaisyUI classes directly
<span class="loading loading-spinner loading-md"></span>

Props

Prop Type Default Description
type string 'spinner' 'spinner' | 'dots' | 'ring' | 'ball' | 'bars' | 'infinity'
size string 'md' 'xs' | 'sm' | 'md' | 'lg'
color string - Color utility class e.g. 'text-primary'
