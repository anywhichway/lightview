Including DaisyUI

DaisyUI and Tailwind polute the global CSS space a redefine things like H1. By default Lightview componetns isolaten DaisyUI and Tailwind styles to shadowDOMS. If you would like to use DaisyUI and Tailwind styles outside of shadowDOMS, add DaisyUI and Tailwind to your HTML:

<!-- DaisyUI + Tailwind CSS (CDN) -->
<link href="https://cdn.jsdelivr.net/npm/daisyui@5" rel="stylesheet" type="text/css" />
<link href="https://cdn.jsdelivr.net/npm/daisyui@5/themes.css" rel="stylesheet" type="text/css" />
<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>

Using with Lightview

Two ways to use DaisyUI styles:

1. JavaScript Components

Import Lightview components that handle DaisyUI classes:

import { Button, Card, Badge } from './components/index.js';

const { $ } = Lightview;

$('#app').content(
    Card({ shadow: 'md' },
        Card.Body({},
            Badge({ color: 'success' }, 'New'),
            Button({ color: 'primary' }, 'Buy Now')
        )
    )
);

2. DaisyUI Classes

Use DaisyUI classes directly with Lightview tags:

const { tags, $ } = Lightview;
const { div, button, span } = tags;

$('#app').content(
    div({ class: 'card shadow-md' },
        div({ class: 'card-body' },
            span({ class: 'badge badge-success' }, 'New'),
            button({ class: 'btn btn-primary' }, 'Buy Now')
        )
    )
);

Registering Styles

Components like Card allow you to inject custom styles directly into their Shadow DOM using the styleSheets prop. This allows you to pass an array of CSS file URLs which LightviewX will fetch, cache, and adopt.

You might want to do this to:

  • Create self-contained custom themes that don't leak out (styles are encapsulated).
  • Reuse the same CSS file across multiple isolated components efficiently.
  • Override DaisyUI defaults for specific component instances without using inline styles.

Example

// 1. Create a CSS file (e.g., /styles/card-theme.css)
/* 
    :host { --card-bg: #f0f; } 
    .card { background: var(--card-bg); }
*/

// 2. Pass the URL to the component
$('#app').content(
    Card({ 
        useShadow: true,
        // LightviewX handles fetching and caching automatically
        styleSheets: ['/styles/card-theme.css'] 
    },
        Card.Body({},
            Card.Title({}, 'Themed Card'),
            tags.p({}, 'This card uses encapsulated styles from a file.')
        )
    )
);

Theme Controller

Add a theme switcher to your app:

import ThemeController from './components/actions/theme-controller.js';

// Toggle between light and dark
ThemeController({ type: 'toggle' })

// Or dropdown with multiple themes
ThemeController({ 
    type: 'dropdown', 
    themes: ['light', 'dark', 'cupcake', 'cyberpunk'] 
})
